# Portafolio Personal

Portfolio profesional desarrollado con [Astro](https://astro.build).

## Tecnologías

- **Framework:** Astro 5.x
- **Package Manager:** Bun

## Comandos

```bash
# Instalar dependencias
bun install

# Iniciar servidor de desarrollo
bun run dev

# Compilar para producción
bun build

# Previsualizar build
bun preview
```
